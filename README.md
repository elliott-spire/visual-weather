# Usage

### URL Parameters

	lat (required)

	lon (required)

	token (required)

	name (not currently used)


### Example

https://elliott-spire.github.io/visual-weather/?name=Dotsero&lat=39.66061&lon=-107.0350&token=[YOURTOKEN]
